# PostHog Feature Flags Skill

Manage and query feature flags.

## Triggers
- User asks about feature flags
- User wants to check flag status
- User mentions "feature flag", "A/B test"

## Tools Available

### posthog_list_flags
List all feature flags.

### posthog_get_flag
Get details of a specific flag.

## Important Notes
- Don't modify flags without explicit approval
- Report any unexpected flag states
