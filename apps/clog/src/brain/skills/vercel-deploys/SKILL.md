# Vercel Deployments Skill

Manage and query Vercel deployments.

## Triggers
- User asks about deployments
- User wants to trigger a deployment
- User mentions "deploy", "preview", "production"

## Tools Available

### vercel_list_deployments
List recent deployments.

### vercel_trigger_deploy
Trigger a new deployment (requires approval).

## Important Notes
- Never deploy without explicit operator approval
- Report deployment failures immediately
